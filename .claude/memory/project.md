# Frankfurt la Torre 2 - Web

## Descripció
Landing page moderna per Frankfurt la Torre 2 (Pineda de Mar) amb integració SmartMenu Agora per gestionar comandes online amb recollida al local.

## Stack
- **Framework:** Astro (components reutilitzables, fàcil escalar)
- **Estils:** CSS/Tailwind
- **Deploy:** A decidir (Vercel/Netlify recomanat)
- **Backend:** Cap (SmartMenu Agora gestiona comandes)

## Usuaris
Clients del restaurant que volen fer comandes per recollir al local.

## Funcionalitats MVP
1. Hero amb CTA "Fer Comanda" ben visible
2. Secció productes estrella (6 destacats)
3. Secció "Nosaltres" (producte local)
4. Ubicació + mapa + contacte
5. Footer amb xarxes socials
6. Botó flotant CTA sempre visible
7. Integració SmartMenu Agora (link extern)

## Disseny
- **Colors:**
  - Groc mostassa: #E8A936

- **Fonts:**
